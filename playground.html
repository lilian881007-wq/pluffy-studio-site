INTERACTIVE ARCHIVE

遊戲與互動作品

這裡收錄過去參與製作的遊戲企劃與互動練習。雖然與現在的設計服務風格不同，但也記錄了角色、介面、動態與體驗設計的養分。

GAME 01

瓦爾格 Wagger

以夜色城市、角色行動與戰鬥節奏為核心的遊戲企劃。

GAMEPLAY PREVIEW

CHARACTER / UI / ACTION

在霓虹城市裡展開的追逐與攻防

作品以暗色都市與高反差霓虹作為視覺基調，透過角色剪影、動作張力與介面資訊，營造偏動作遊戲的節奏感。頁面可放置宣傳影片、角色設定、介面截圖與製作紀錄。

負責內容
角色視覺整理、UI 草圖、宣傳圖與作品頁編排
呈現重點
世界觀氛圍、角色動作、戰鬥介面與遊戲影片

GAME 02

記憶迴廊 Memory Loop

以關卡探索與道具提示為主的互動原型。

PROTOTYPE PLAY

PUZZLE / FLOW / PROTOTYPE

用小型機制測試玩家的理解路徑

這類作品重點不在完整商業化，而是在測試操作提示、路線引導與回饋節奏。透過簡單的關卡流程，觀察玩家是否能理解目標、辨識可互動物件並完成解謎。

設計重點
道具提示、互動回饋、關卡流程與 UI 引導
可放素材
遊玩影片、流程圖、場景截圖或 early demo

GAME 03

角色對話互動

以角色表情、對話框與選單互動為主的練習作品。

INTERACTION TEST

CHARACTER / DIALOGUE / MOTION

讓角色透過對話與介面被記住

以角色立繪、台詞框、選項按鈕與小動態為核心，練習如何讓玩家在短時間內理解角色個性與互動節奏。適合收錄角色設定、分鏡、對話畫面與動態測試。

呈現重點
角色表情、對話 UI、選項流程與情緒節奏
延伸紀錄
可放團隊專案、課堂作品或未完成原型

TEAM

團隊與專案背景

這些作品多半來自過去的課堂專案、團隊練習或個人原型。頁尾可放團隊 Logo、成員分工、使用工具與製作時間，讓作品脈絡更完整。
